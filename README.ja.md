# KaraokeQ — スマートカラオケ予約キュー管理ツール

[![English](https://img.shields.io/badge/lang-English-blue.svg)](./README.md)
[![简体中文](https://img.shields.io/badge/lang-简体中文-red.svg)](./README.zh-CN.md)
[![日本語](https://img.shields.io/badge/lang-日本語-green.svg)](./README.ja.md)

あらゆるデバイスで使える、モダンなウェブ版カラオケ点歌・キュー管理ツールです。

## 機能

- **ディスプレイモード** — テレビやプロジェクターでのフルスクリーン動画/音声プレーヤー
- **リモコン操作** — 同じネットワーク上の任意のスマホやタブレットから曲の追加・削除・並べ替えが可能
- **YouTube 対応** — 完全な API 制御と自動再生で YouTube 動画を自動再生
- **マルチフォーマット** — Bilibili 埋め込み、直リンクの MP4/MP3 ファイル、汎用 iframe にも対応
- **リアルタイム同期** — 接続中のすべてのデバイスが即座に同期
- **多言語対応** — English、简体中文、繁體中文、日本語、한국어、Español、Français、Deutsch
- **音量コントロール** — 任意のリモートデバイスから調整可能
- **ドラッグで並べ替え** — キュー内の曲をドラッグして再生順を変更
- **QR コードで参加** — ディスプレイに QR コードを表示し、スマホから簡単にアクセス

## 使い方

### カラオケセッションを開始する

1. **ディスプレイ機器**（テレビ/プロジェクター/ノート PC）でサイトを開く
2. **Create Room（ルーム作成）** をクリック — 6 文字のルームコードと QR コードが表示されます
3. ディスプレイがフルスクリーンプレーヤーモードに切り替わり、ルームコードと QR コードを表示します

### 曲を追加する（任意のスマホから）

1. ディスプレイの QR コードをスキャンするか、`remote.html?room=XXXXXX` を開く
2. またはサイトを開いて **Join Room（ルーム参加）** をクリックし、コードを入力する
3. **+** ボタンをタップ → URL を貼り付け → **Add to Queue（キューに追加）** をタップ

**対応している URL の種類:**
| 種類 | 例 |
|------|---------|
| YouTube | `https://youtube.com/watch?v=...` または `https://youtu.be/...` |
| YouTube Shorts | `https://youtube.com/shorts/...` |
| Bilibili | `https://bilibili.com/video/BVxxx` |
| 直リンク動画 | `https://example.com/song.mp4` |
| 直リンク音声 | `https://example.com/song.mp3` |
| 汎用 iframe | その他の任意の URL（自動再生は制限あり） |

### キーボードショートカット（ディスプレイ機器上）

| キー | 動作 |
|-----|--------|
| `Space` | 再生 / 一時停止 |
| `→` | 次の曲へスキップ |
| `↑` / `↓` | 音量アップ / ダウン |
| `Q` | キューパネルの表示切替 |

## 自動再生の動作

| プレーヤー種別 | 曲終了時に自動再生？ |
|-------------|------------------------------|
| YouTube | ✅ はい（YouTube IFrame API 経由） |
| HTML5 動画/音声 | ✅ はい（`ended` イベント経由） |
| Bilibili / iframe | ❌ 手動スキップが必要 |
